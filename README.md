# Netra AI Vision Interface

Netra is an AI-powered assistance system designed for the visually impaired. It utilizes various engines for depth perception, behavior analysis, emotion recognition, and more.

## Features
- **Depth Perception**: Real-time depth mapping for obstacle avoidance.
- **Behavior Analysis**: Understanding environmental context and human behaviors.
- **Emotion Recognition**: Identifying emotions of people in the environment.
- **Voice Integration**: Hands-free interaction and auditory feedback.

## Project Structure
- `backend/`: Python-based AI engines and core logic.
- `js/`: Frontend interaction and visualization logic.
- `diag_midas_local.py`: Diagnostic tools for the MiDaS depth model.
- `test_all_engines.py`: Comprehensive test suite for AI engines.
